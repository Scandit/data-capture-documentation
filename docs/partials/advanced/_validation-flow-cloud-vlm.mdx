### Cloud-Based Scanning

When the on-device models are not able to read a field, you can enhance scanning with cloud-based solutions. Adaptive Recognition uses a cloud-based Vision Language Model (VLM) to read label data from a captured image, providing a seamless fallback that avoids requiring the user to type data manually.

Enable Adaptive Recognition by setting the mode to `.auto` on the label definition:
