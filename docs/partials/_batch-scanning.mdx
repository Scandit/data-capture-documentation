Scandit's unparalleled batch scanning capabilities (MatrixScan) are available as a low-level API as well as use-case optimized workflows with built-in camera controls, dialed-in performance and extensively user-tested UI.

## Low-level API

<p align="center">
  <img src="/img/batch-scanning/MatrixScanBatch.gif" alt="MatrixScan Batch Workflow" />
</p>

Batch scanning, implemented via the `BarcodeBatch` API, enables you to build applications and workflows for capturing and interacting with multiple barcodes simultaneously.

**Available For:** [iOS](/sdks/ios/matrixscan/intro.md), [Android](/sdks/android/matrixscan/intro.md), [React Native](/sdks/react-native/matrixscan/intro.md), [Flutter](/sdks/flutter/matrixscan/intro.md), [Web](/sdks/web/matrixscan/intro.md), [Cordova](/sdks/cordova/matrixscan/intro.md), [Capacitor](/sdks/capacitor/matrixscan/intro.md), Xamarin ([iOS](/sdks/xamarin/ios/matrixscan/intro.md), [Android](/sdks/xamarin/android/matrixscan/intro.md), [Forms](/sdks/xamarin/forms/matrixscan/intro.md)), .NET ([iOS](/sdks/net/ios/matrixscan/intro.md), [Android](/sdks/net/android/matrixscan/intro.md))
