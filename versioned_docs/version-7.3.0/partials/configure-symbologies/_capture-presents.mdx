### Symbology Presets by Industry

:::warning
The capture settings presets in this section are currently only available for single scanning modes (Barcode Capture, SparkScan) and are **not available** for MatrixScan.
:::

An alternative to enabling your desired symbologies individually in your capture settings, you can use a predefined industry vertical `CapturePreset`. 

The available presets and the corresponding symbologies enabled by each is listed below.

:::tip
You can still enable or disable individual symbologies as needed after creating your capture settings using one the presents.
:::

| Preset | Symbologies Enabled |
| ------ | ------------------- |
| TRANSPORT | Code128<br/>QR Codes<br/>Code 39<br/>Data Matrix<br/>EAN13_UPCA<br/>Interleaved 2-of-5<br/>Aztec<br/>EAN8<br/>PDF417<br/>UPCE |
| LOGISTICS | Code128<br/>QR Codes<br/>Code 39<br/>Data Matrix<br/>EAN13_UPCA<br/>Interleaved 2-of-5<br/>Codabar<br/>EAN8<br/>PDF417<br/>UPCE |
| RETAIL | EAN13_UPCA<br/>Code128<br/>QR Codes<br/>Code 39<br/>EAN8<br/>Data Matrix<br/>Interleaved 2-of-5<br/>UPCE<br/>GS1 Databar<br/>GS1 Databar Expanded |
| HEALTHCARE | Code128<br/>Data Matrix<br/>QR Code<br/>EAN13_UPCA<br/>Code39<br/>MicroPDF417<br/>Interleaved 2-of-5<br/>MSI Plessey<br/>EAN8 |
| MANUFACTURING | Code128<br/>Data Matrix<br/>Code 39<br/>QR Codes<br/>EAN13_UPCA<br/>Interleaved 2-of-5<br/>PDF417<br/>UPCE<br/>EAN8 |
